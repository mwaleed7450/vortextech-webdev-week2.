# Portfolio Website

Personal portfolio site built as part of a weekly practice project. Simple, responsive, and includes a working contact form with validation plus a dark/light mode toggle.

## What's inside

- **index.html** — page structure (About, Skills, Contact sections)
- **style.css** — styling, including CSS variables for the dark/light theme
- **script.js** — theme toggle logic + form validation

## Features

- Dark/light mode toggle (saved via a button in the top-right corner)
- Responsive layout that works on mobile and desktop
- Contact form with client-side validation (name, email format, message required)
- Success message shown after valid submission
- No frameworks — plain HTML, CSS, and JavaScript
